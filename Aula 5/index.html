Trecho de código fonte

Neste exemplo vou desmonstrar um trecho de código em Python


import pandas as pd

                # Exemplo de dados
                data = pd.DataFrame({'valor': [10, 12, 12, 13, 100, 15, 16, 17, 18, 200]})
                
                # Calculando o primeiro (Q1) e o terceiro quartil (Q3)
                Q1 = data['valor'].quantile(0.25)
                Q3 = data['valor'].quantile(0.75)
                
                # Calculando o intervalo interquartil (IQR)
                IQR = Q3 - Q1
                
                # Definindo os limites inferior e superior para detectar outliers
                limite_inferior = Q1 - 1.5 * IQR
                limite_superior = Q3 + 1.5 * IQR
                
                # Removendo os outliers
                data_sem_outliers = data[(data['valor'] >= limite_inferior) & (data['valor'] <= limite_superior)]
                
                print(data_sem_outliers)
